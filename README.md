# Physics Simulator

A 3D physics engine built with Java and LWJGL, featuring real-time physics simulation with collision detection and response.

## Version Information

- **Java**: 24
- **LWJGL**: 3.3.6 (Latest stable)
- **JOML**: 1.10.8 (Latest stable)
- **Maven**: 3.6.3+
- **Last Updated**: January 2025
