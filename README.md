# Solar System

A 3D solar system simulation built with Java and LWJGL. Features accurate orbital dynamics and customizable viewing angles.

## Version Information

- **Java**: 24
- **LWJGL**: 3.3.6 (Latest stable)
- **JOML**: 1.10.8 (Latest stable)
- **Maven**: 3.6.3+
- **Last Updated**: July 2025
